PDFTool

A lightweight Windows tool that automatically merges a multi-page Bill of Materials (BOM) PDF with a single insert page.

Download PDFTool (Windows)

What does PDFTool do?

PDFTool creates a new PDF where each page of your Bill of Materials is immediately followed by the same insert page. 

Page 1  – BOM page 1
Page 2  – Insert page
Page 3  – BOM page 2
Page 4  – Insert page
Page 5  – BOM page 3
Page 6  – Insert page

Features

  • Drag & drop interface
  • Automatic page alternation (BOM / insert)
  • Offline usage – no internet required
  • Portable single EXE (no installation)
  • Windows 10 & Windows 11

Privacy & Security

PDFTool works completely offline. No data is collected, stored, or transmitted.

Antivirus notice

To keep this project 100% free, the executable is not code-signed. Some antivirus software or Windows Defender may show a warning.

The source code is publicly available on GitHub.
